HeritageGraph Knowledge Graph — Licensing
==========================================

The HeritageGraph KG combines data from multiple open sources. To keep
licensing unambiguous, each source is published as a SEPARATE named-graph file
under kg/ and is governed by the license of its source. Redistribution must
preserve the attribution and license terms below.

--------------------------------------------------------------------------
kg/wikidata.ttl
  Source : Wikidata (https://www.wikidata.org/)
  License: Creative Commons CC0 1.0 Universal (public domain dedication)
           https://creativecommons.org/publicdomain/zero/1.0/

kg/unesco.ttl
  Source : UNESCO World Heritage Centre (https://whc.unesco.org/)
  Note   : Factual World Heritage inscription data. Attribute the UNESCO
           World Heritage Centre.

kg/intangible.ttl
  Source : Curated by CAIR-Nepal from Wikidata (CC0), UNESCO, and published
           scholarship (Slusser; Toffin; Gellner; Locke) and the Nepal
           Heritage Documentation Project.
  License: Creative Commons Attribution 4.0 International (CC BY 4.0)
           https://creativecommons.org/licenses/by/4.0/

kg/osm.ttl
  Source : OpenStreetMap (https://www.openstreetmap.org/)
  License: Open Database License (ODbL) v1.0
           https://opendatacommons.org/licenses/odbl/1-0/
  Attribution: "© OpenStreetMap contributors"
  IMPORTANT: ODbL is a share-alike license. Any database derived from
           kg/osm.ttl (including a combined graph that incorporates it) must
           be made available under ODbL with the above attribution. The
           CC0/CC-BY layers (kg/wikidata.ttl, kg/unesco.ttl, kg/intangible.ttl)
           may be used independently of the OSM layer.

kg/crosswalk.ttl
  Mixed-provenance linking layer (owl:sameAs / skos:altLabel). Each link
  inherits the license of the source it references.

kg/intangible_crosswalk.ttl
  Mixed-provenance linking layer for the intangible heritage subgraph
  (owl:sameAs / extra typing). Each link inherits the license of the source
  it references.
